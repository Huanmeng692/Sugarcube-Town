具体是怎么做呢？
首先我们需要给开场画面增加一个点击后，将这个画面移除的效果。
这有两个难点，一个是如何获得这个div元素，以及如何操作它。
我们一步一步的来讲。

获得HTML元素，需要用到JS自带的DOM方法，而要做到这一点，我们首先需要这个元素存在于页面上
也就是等待浏览器将页面加载完成，不然就会报错，很烦的
而Sugarcube提供了一个很方便的功能来确保这一点。那就是<<done>>宏，字面意义，这个宏的作用是
将内部包裹的代码延迟到页面完全加载完毕之后执行。
而执行的内容，是Javascript脚本，我们又需要用到另一个宏，也就是<<script>>
这个宏内部包裹的代码，浏览器会视作纯Javascript来执行，所以我们第一步要写的内容是：

// 首先我们要选中这个DIV，将它保存在js的变量内。
// const 声明一个变量的引用
// 给它赋值成什么呢，这我们就要用到JS的一个自带方法了
// 这个方法可以根据元素的ID，获取这个元素本身
const vars = document.getElementById("opening");
// 然后我们需要给这个元素增加一个点击事件
// 同样会用到dom方法，不过是另一个
// 它需要两个参数，一个是事件触发的类型
// 这里我们选择的是点击事件
// 然后输入第二个参数
// 这是JS的函数的写法
vars.addEventListener("click", function () {
	// 在这里写函数的内容
	// 就比如我们想要的这个效果
	// 我们用this关键字代表函数所在的元素本身，也就是上面的vars变量引用的HTML元素
	// 这个HTML元素有很多自带的方法，其中一个是remove,也就是删除
	// 别忘了方法的后面需要加上英文括号，代表你使用这个方法，而不是引用它
	this.remove();
})
